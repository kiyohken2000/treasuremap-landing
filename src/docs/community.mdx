# FGO コミュニティ

## ユーザー間の連絡手段にはPINE proを推奨します。

[PINE pro](https://pinepro.ml/)はFind Good One開発者が開発・運営するメッセージングアプリです。FGOユーザー間の連絡にはPINE proをご利用ください。

開発者からのお知らせはPINE proを通じて行います。また、アプリのフィードバックにもご利用ください。

![](./img23.jpg)

### 主な機能

- メールアドレスによるサインアップとログイン
- グループチャット
- オープンチャット機能
- 画像の送信
- 新着メッセージの通知
- 広告なし、完全無料
- ユーザーデータのトラッキングなし

### アカウント

FGOとPINE proの間でユーザー情報は共有していません。アプリごとにアカウントを作成してご利用ください。
