# Introduction

**Find Good One**はスマートフォンのバックグラウンド位置情報取得機能とジオフェンスを使った宝探しアプリです。

ユーザーはマップ上の任意の座標に架空の宝箱を自由に設置できます。宝箱には、発見した人へのメッセージや画像を設定できます。

他のユーザーが設置した宝箱に接近するとスマートフォンに宝箱を発見した事が通知されます。宝箱を取得することを選ぶと自分のアイテムリストにそれが追加されます。取得した宝箱はいつでも見返すことができます。


## Features

- ⚛️ バックグラウンド位置情報の取得
- 📝 画像アップロード
- 🛣 メールアドレスによるログインとサインアップ
- 📱 プッシュ通知(ローカル通知とリモート通知)
- 🥇 ジオフェンス

### 位置情報の取得と通知を許可にしてご利用ください。

- `Appの使用中は許可`
- `"常に許可"に変更`
